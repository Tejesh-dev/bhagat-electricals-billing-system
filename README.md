How to Use
Inventory Setup:

Navigate to the Admin tab.

Enter your product name and price.

Select the appropriate category and click "Save to Inventory."

Creating a Bill:

Go to the Shop tab.

Browse items or filter by category.

Click "Add" on the items required.

Checkout:

Go to the Cart tab.

Input the Customer Name.

Adjust quantities if necessary.

Click "Generate Final Bill" to download the premium PDF.

History:

Access the History tab to view or re-download any previous bill.
